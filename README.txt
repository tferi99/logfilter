Description
-----------
It filters text-files from input directory and writes matching lines into files (per filter)
of output directory. Filters cxan be specified in property file:
	
	filter_name1=pattern1
	filter_name2=pattern2
	...
	filter_nameN=patternN


Usage
-----
- call build.bat

- call run.bat

	Input directory: data/in
	Input directory: data/out
